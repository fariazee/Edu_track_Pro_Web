    About Us

    Welcome to EduPredict

    EduPredict is an educational analytics platform that uses machine learning to help schools and analysts predict student outcomes, track attendance, and understand demographic trends.

    Built with Django and scikit-learn, EduPredict offers role-based access for analysts and students—giving each user the right tools to make informed, data-driven decisions.
